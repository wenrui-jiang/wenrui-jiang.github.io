北京合力云通科技有限公司 - Java 开发工程师（实习）

任职时间

2022.12 - 2023.03

项目名称

A/B 测试平台开发

项目简介

A/B 测试是一种通过比较网页或应用多个版本来评估效果的方法，常用于页面迭代、个性化内容展示和用户行为分析。该项目围绕实验配置、任务调度、结果收集和反馈报告生成展开。

技术栈

Spring Cloud, MyBatis, MySQL, RabbitMQ, XXL-JOB

工作内容

  • 后端接口开发：根据用户在前端配置的时间参数，调用 xxl-job 微服务定时启动 A/B 测试实验。
  • 任务状态管理：支持对运行中的实验任务进行暂停、参数修改和重启。
  • 实验结果处理：收集实验结果，并将数据输入多臂老虎机算法，计算新版本页面的流量分配策略。
  • 报告与通知：调用网页转 PDF 接口生成实验反馈报告，并通过邮件接口发送给客户。

项目收获

  • 熟悉了实验平台的后端业务流程和任务调度方式。
  • 参与了从实验创建、执行、结果处理到报告反馈的完整链路。
  • 加深了对 Spring Cloud、MyBatis、RabbitMQ 和定时任务系统的工程实践理解。
