# Project Editing Guide

This is a multilingual personal website. Keep content changes synchronized across all language versions.

## Required Rule

When changing any public-facing content, navigation item, section meaning, contact method, article, experience entry, or portfolio/article data, update every language version in the same change.

Current languages:

- English: `content/english`
- Chinese: `content/Chinese`
- Spanish: `content/spanish`
- French: `content/french`
- German: `content/german`
- Arabic: `content/arabic`
- Latin: `content/latin`

If a translation is not final, add a conservative translated draft rather than leaving the language missing or showing stale content.

## Site Structure

- `post/` is used as the technical Hugo section for work and internship experiences. The user-facing label is `Experience` / `经历`.
- `portfolio/` is used as the technical Hugo section for article/research cards. The user-facing label is `Articles` / `文章`.
- Avoid template/demo content. Do not reintroduce Hugo sample posts, placeholder projects, lorem ipsum text, or theme demo pages.

## Privacy

Use only public-safe resume details. Do not publish private identifiers, private addresses, or unnecessary personal data.
